# Dcat Admin Extension

# 简介
本扩展是一个接口文档类扩展，主要功能是管理api接口，并可以分组和对接口进行测试

# 安装与使用
安装：可以下载仓库成zip包,在DcatAdmin 扩展页面，使用本地安装即可。
使用：在表单录入信息后保存，左则导航栏就会增加一项接口。如果接口地址相同，则会更新。
